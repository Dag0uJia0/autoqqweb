# autoqqweb
自动打站的skill
# 创建该项目的目标为更轻松的实现自动渗透
## 安装
## Claude Code
```
git clone https://github.com/Dag0uJia0/autoqqweb.git ~/.claude/skills/autoqqweb
```
## 卸载
### Linux
```
rm -rf ~/.claude/skills/autoqqweb
rm -rf ~/.claude/skills/test-skill
```
### Win
```
rm -Recurse -Force ~/.claude/skills/autoqqweb
rm -Recurse -Force ~/.claude/skills/test-skill
```
## 更新
### Linux
```
cd ~/.claude/skills/autoqqweb && git pull
```
### Win powershell
```
cd $env:USERPROFILE\.claude\skills\autoqqweb
git pull
```
